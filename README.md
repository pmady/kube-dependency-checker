# Kube Dependency Checker

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

A tool to check and analyze Kubernetes dependencies and version compatibility.

## Overview

Kube Dependency Checker helps you identify and manage dependencies in your Kubernetes clusters and manifests. It provides insights into version compatibility, deprecated APIs, and upgrade paths.

## Features

- Check Kubernetes API version compatibility
- Identify deprecated APIs in manifests
- Analyze Helm chart dependencies
- Validate version constraints
